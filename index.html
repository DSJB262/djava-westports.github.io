Setup Instructions

  1. Open Google Sheets → create a new spreadsheet
  2. Click Extensions → Apps Script
  3. Delete all code → paste contents of apps-script.gs → Ctrl+S
  4. Click Deploy → New deployment → Web app → Execute as Me → Anyone → Deploy
  5. Copy the URL → paste above → Save & Connect
