Team / Assignees

Email → RE-ID Auto-Ticket

Scans Gmail for unread emails containing ##RE-XXXXX## in subject or body.
• New request created with ID: ##RE-XXXXX## → creates a new ticket automatically.
• Request with ID: ##RE-XXXXX## … has been Closed → closes the matching ticket.

User Management

Admin-only. Manage who can access this tracker.

SLA Targets (days per stage)

Max days a ticket should stay in each stage. Age badge turns red with ⚠ when exceeded. Requires Updated Date field from your GAS backend.

Daily Developer Reminder

Sends each developer an email at the chosen hour listing their open tickets. Requires sendDailyReminders action in your GAS backend (see GAS code below).

Setup Instructions

  1. Open Google Sheets → create a new spreadsheet
  2. Click Extensions → Apps Script
  3. Delete all code → paste contents of apps-script.gs → Ctrl+S
  4. Click Deploy → New deployment → Web app → Execute as Me → Anyone → Deploy
  5. Copy the URL → paste above → Save & Connect
